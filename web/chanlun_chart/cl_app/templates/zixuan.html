使用说明：

添加自选组，在相同市场不允许有相同名称的自选组

删除自选组，默认自选组 "我的关注" 是不允许删除的

导入/导出操作，先选择要进行导入或导出到的 自选组，然后再点击对应的操作

导入/导出规则说明：

支持的文件格式：TXT、CSV（UTF-8 编码）

导出格式：CSV 文件，每行格式为 代码,名称

导入格式：每行一条记录，支持以下两种写法：

  • 仅股票代码（如：SZ.000001）—— 名称将留空
  • 股票代码和名称（如：SZ.000001,平安银行）

代码格式兼容规则（各市场）：

  • 沪深A股：支持 600519 / SH.600519 / SHSE.600519 三种格式
  • 国内期货：支持 RB0000 / RB.SHF 格式
  • 纽约期货：支持 GC / GC.NYMEX 格式
  • 数字货币：支持 BTC / BTC-USDT.SWAP 格式
  • 其他市场：支持带或不带市场后缀的代码格式

CSV 表头处理：如果文件第一行以 stock_code 或 code 开头，会自动作为表头跳过

注意事项：

  • 导入时，如果代码在当前市场无法匹配到唯一标的，该行将被跳过
  • 空行会自动忽略
  • 导入是追加模式，不会覆盖已有的自选股
导入文件示例（CSV/TXT 内容） 
SH.600000,浦发银行
SZ.000001,平安银行
SH.600519,贵州茅台
SZ.000002,万科A
