要进行 QMT 实盘交易，你需要关注三个部分：**运行脚本**、**交易实现**、**策略逻辑**。

### 1. 实盘交易怎么用？（配置与运行）

实盘交易的入口不是你直接修改 `src/chanlun/trader` 下的文件，而是运行 `script/trader/` 目录下的专用脚本。

以股票为例，对应的脚本是：`script/trader/run_qmt_stock.py`

#### 第一步：配置运行脚本
打开 [run_qmt_stock.py](file:///d:/quantitative/chanlun-pro/script/trader/run_qmt_stock.py) (或 `run_qmt_futures.py` / `run_qmt_option.py`)，修改以下关键参数：

```python
# 1. 定义监控的代码列表 (支持动态添加，也可以在这里写死)
run_codes = ['SH.600519', 'SZ.000001']

# 2. 运行周期 (通常至少包含一个高级别)
frequencys = ['5m', '30m']

# 3. 策略对象 (决定什么时候买卖)
# 默认使用的是线段买卖点策略，你可以替换为你自己写的策略类
STR = StrategyXDMMD() 
```

#### 第二步：启动交易
确保你的 **MiniQMT 客户端** 已经启动并登录（且开启了 Python API 支持）。
然后在终端运行：

```powershell
python script/trader/run_qmt_stock.py
```
*(如果是期货或期权，请运行对应的 `run_qmt_futures.py` 或 `run_qmt_option.py`)*

### 2. 缠论买卖哪里触发？（策略与信号）

触发买卖的逻辑**不**在 `trader_qmt_*.py` 文件中，而是在**策略文件**中。

1.  **数据流向**：
    `run_qmt_stock.py` 会启动一个循环，不断从 QMT 获取最新行情 -> 计算缠论数据 -> 调用策略检查。

2.  **触发位置**：
    在 `run_qmt_stock.py` 中，有这样一行代码：
    ```python
    TR.set_strategy(STR)
    ```
    这里的 `STR` 就是策略实例（例如 [StrategyXDMMD](file:///d:/quantitative/chanlun-pro/src/chanlun/strategy/strategy_xd_mmd.py)）。

3.  **核心逻辑**：
    打开 [strategy_xd_mmd.py](file:///d:/quantitative/chanlun-pro/src/chanlun/strategy/strategy_xd_mmd.py) (或你自定义的策略)，找到 `open` 和 `close` 方法：
    *   **open (开仓)**：当代码检测到缠论买点（如 `xd.line_mmds()` 出现买点）时，返回一个 `Operation` 对象。
    *   **close (平仓)**：当代码检测到卖点或止损条件时，返回平仓信号。

    **信号传递链**：
    `Strategy.open()` 返回买入信号 -> `BackTestTrader.run()` 接收信号 -> 调用 `QMTTraderStock.open_buy()` -> 发送订单到 QMT 客户端。

### 3. 文件功能说明

你提到的三个文件是**交易执行层**，负责将策略生成的“信号”转化为实际的“QMT 订单”。

*   [trader_qmt_stock.py](file:///d:/quantitative/chanlun-pro/src/chanlun/trader/trader_qmt_stock.py)：
    *   **功能**：实现了股票的实盘下单、撤单、持仓查询。
    *   **修改建议**：一般不需要修改，除非你需要改变下单价格模式（如改为限价单）或调整资金分配逻辑（如 `open_buy` 中的 `amount` 计算逻辑）。
    *   **注意**：第 92 行 `self.acc = StockAccount("8886136661")` 需要**替换为你自己的 QMT 资金账号**。

*   [trader_qmt_futures.py](file:///d:/quantitative/chanlun-pro/src/chanlun/trader/trader_qmt_futures.py) / [trader_qmt_option.py](file:///d:/quantitative/chanlun-pro/src/chanlun/trader/trader_qmt_option.py)：
    *   同理，分别对应期货和期权。记得修改其中的 `StockAccount` 账号。

### 总结操作步骤

1.  **修改账号**：去 `src/chanlun/trader/trader_qmt_*.py` 中把 `StockAccount("...")` 换成你的真实账号。
2.  **配置策略**：在 `script/trader/run_qmt_*.py` 中设置你要监控的 `run_codes` 和使用的策略。
3.  **启动运行**：运行 `python script/trader/run_qmt_*.py`。
